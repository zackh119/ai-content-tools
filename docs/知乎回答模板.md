## 回答模板（以问题1为例）

> 问题：有没有好用的AI文案工具推荐？

**回答：**

试过不少AI写作工具，说一个我自己在用的。

先说一下我的需求：主要是写小红书，偶尔写抖音脚本和公众号。市面上的工具要么太贵（动不动月付99），要么生成的内容AI味太重，一眼就能看出来。

后来我一咬牙自己写了一个。

流程很简单：
1. 输入主题
2. AI先出5个标题让你选
3. 选好了生成全文
4. 不满意还能精修

最满意的一点是它内置了去AI味系统，"值得注意的是""综上所述"这类词全部自动避开。生成出来的文案口语化、有情绪，基本看不出是AI写的。

而且它不只写小红书——
切换tab可以写抖音脚本（自动分画面/口播/字幕）
还能写公众号长文（带小标题分段）

免费，链接在这里：
https://zackh119.github.io/ai-content-tools/

需要自己准备一个API Key
（在DeepSeek官网注册就有，10块钱能用好几个月）

有需要的可以试试，不好用来骂我。
